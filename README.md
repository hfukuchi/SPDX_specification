# SPDX_specification
Unofficial Japanese translation of SPDX specification

## SPDX仕様(Version 2.1)の非公式日本語訳を行っています

### Software Package Data Exchange® (SPDX®) はソフトウェア パッケージのライセンス情報やメタデータを企業や組織間で共有するための規格です
